# Essentia Demo

Audio analysis demo using [@siteed/react-native-essentia](https://github.com/deeeed/audiolab/tree/main/packages/react-native-essentia) — MFCC, spectral features, key detection, and more.

Part of the [audiolab](https://github.com/deeeed/audiolab) monorepo.

## Setup

```bash
cd apps/essentia-demo
yarn install
yarn start
```
