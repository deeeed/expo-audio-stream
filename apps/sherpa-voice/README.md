# Sherpa Voice

On-device speech recognition, text-to-speech, and audio intelligence powered by [sherpa-onnx](https://github.com/k2-fsa/sherpa-onnx). Part of the [audiolab](https://github.com/deeeed/audiolab) monorepo.

[Try the web demo](https://deeeed.github.io/audiolab/sherpa-voice/)

## Setup

```bash
cd apps/sherpa-voice
yarn install
yarn start
```

Requires Git LFS for ONNX models — run `../../scripts/setup-lfs.sh` first.
